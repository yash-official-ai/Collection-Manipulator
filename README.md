# 📚 Student Data Organizer

A simple yet powerful **Python Console-Based Student Management System** that helps manage student records efficiently.  
This project allows users to add, display, update, delete, and organize student information directly from the terminal.

---

## ✨ Features

✅ Add New Student Records  
✅ Display All Students  
✅ Update Existing Student Details  
✅ Delete Student Records  
✅ View Unique Subjects Offered  
✅ User-Friendly Menu Driven Interface  

---

## 🖥️ Console Preview

```bash
===== Welcome to Student Data Organizer =====

1. Add Student
2. Display All Students
3. Update Student
4. Delete Student
5. Display Subjects
6. Exit
```

# 📌 Example Usage

## ➕ Add Student

```bash
Enter your choice: 1

Enter Name: yash
Enter Age: 17
Enter Grade: 12
Enter Subjects separated by comma: statistics,economics,o.c.m,english
Enter Student ID: 12882
Enter Date of Birth: 06/08/2008

Student Added Successfully!
```
